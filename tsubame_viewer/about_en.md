### Supported Platforms

* Windows 11
* Windows 10 (Version 1809 or later required)

## App Overview

* Register a folder with the app to get started
* Supports the entire process from browsing folders to viewing files in the viewer
* View images inside compressed files without extracting them to storage
* Supports two-page spread view (Image Viewer)
* Supports vertical text layout (Novel Viewer)
* Supports display and operation on tablets

## Supported File Formats

* Images
  * .jpg/.png/.bmp/.gif/.tiff/.svg/[.webp](https://apps.microsoft.com/detail/9pg2dk419drg)(*1)/[.avif](https://apps.microsoft.com/detail/9mvzqvxjbq9v)(*1)
* Archives
  * .zip/.cbz/.rar/.cbr/.7z/.cb7/.tar/.pdf
* E-books
  * .epub
* Video
  * .mp4/[.webm](https://apps.microsoft.com/detail/9pg2dk419drg)(*1)/[.hevc](ms-windows-store://pdp/?ProductId=9n4wgh0z6vhq)(*1)

*1 Requires a separate extension to add codecs


## Features

### Image Viewer

* Supports two-page spread view
* Preloads previous and next pages
* Zoom in/out
 
### Novel Viewer

* Supports vertical and horizontal text orientation
* Text adjusts to fit the screen (supports “reflowable layout”)
* Customize fonts, text size, background color, and more
* Jump to specific chapters


### Movie Viewer

* Flip horizontally
* Adjust playback speed
* Loop playback
* Move forward or backward by one frame
* Background playback
* Swipe left or right to move the playback position
* Swipe up or down to adjust the volume

### Features Common to All Viewers

* Bookmark feature
  * Displays the last viewed position. Automatically resumes playback when reopened
* Displays compressed files directly without extracting them to storage
  * Supported by both the Manga Viewer and Novel Viewer
* Displays viewing progress at the bottom of the screen


### App Features

* Supports launching files by file extension association from Explorer and other applications
* Supports opening files and folders via drag-and-drop
* Supports pinning to the Start screen
* Filter view within the displayed folder
* Search across all folders registered in the app
