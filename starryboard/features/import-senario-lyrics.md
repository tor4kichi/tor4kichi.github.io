# シナリオ・歌詞の取り込み

シナリオや歌詞となるテキストファイルを取り込んでカットに変換できる機能です。

## シンプルなシナリオテキスト

拡張子：.txt

セリフごとに空行を置いたテキストファイル。空行までを１カット分として扱います

タイムはゼロ秒として扱います。

```
セリフセリフ

話者名「セリフセリフ

セリフセリフ」
```


## 歌詞 LyRiCs

拡張子：.lrc

一行のタイムとセリフを１カットとして扱います。

次行のタイムとの差分を前カットのタイムとして扱います。

```
[00:00.00] タイトル
[00:01.00] 
[00:01.00] 空行表示
```


## 字幕 WebVTT

拡張子：.vtt

ファイル先頭に「WEBVTT」の文字列を置くと認識します。

タイムの開始と終了の１行とセリフから空行までを１カットとして扱います。

次行のタイムとの差分を前カットのタイムとして扱うため、基本的にタイムの終了は読み飛ばされます。最後のカットのみタイムの開始と終了をカットのタイムとして扱います。

```
WEBVTT

00:09.000 --> 00:11.000
<話者名>セリフセリフセリフセリフ

00:11.000 --> 00:13.000
<話者名>セリフセリフセリフセリフ
```

## 字幕 SubRip Subtitle

拡張子：.srt

先頭にシーケンス番号を与える以外はWEBVTTと同様です。

```
1
00:05:00.400 --> 00:05:15.300
セリフセリフ
セリフセリフ

2
00:05:16.400 --> 00:05:25.300
セリフセリフ
```
