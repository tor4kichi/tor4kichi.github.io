## [Starryboard](/starryboard/about)

## シナリオ・歌詞の取り込みについて

シナリオや歌詞となるテキストファイルを取り込んでカットに変換できる機能です。

Starryboardのアプリ上からはシーン

### シンプルなシナリオテキスト

拡張子：.txt

セリフごとに空行を置いたテキストファイル。

空行までを１カット分として扱います。

```
セリフセリフ

話者名「セリフセリフ

セリフセリフ」
```


### 歌詞 LyRiCs

拡張子：.lrc

```
[00:00.00] タイトル
[00:01.00] 
[00:01.00] 空行表示
```


### 字幕 WebVTT

拡張子：.vtt

ファイル先頭に「WEBVTT」の文字列を置くと認識します。

```
WEBVTT

00:09.000 --> 00:11.000
<話者名>セリフセリフセリフセリフ

00:11.000 --> 00:13.000
<話者名>セリフセリフセリフセリフ
```

### 字幕 SubRip Subtitle

拡張子：.srt

```
1
00:05:00.400 --> 00:05:15.300
セリフセリフ
セリフセリフ

2
00:05:16.400 --> 00:05:25.300
セリフセリフ
```
