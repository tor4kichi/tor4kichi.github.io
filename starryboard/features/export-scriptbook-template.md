## [Starryboard](/starryboard/about)

## 台本エクスポートをカスタマイズ

Starryboard の台本エクスポートはひな型（テンプレート）を使って.docxファイルの出力内容を制御しています。

このページはカスタムなテンプレートを用意したい方向けに簡易ですがご説明するものです。

### スクリプト知識のある方向けのQuick Start

テンプレート記述方法：[DocxTemplaterの定義](https://github.com/tor4kichi/DocxTemplater#quick-reference-examples)

展開元となる変数はJSONエクスポートの結果がそのまま使えます。

## テンプレート処理

文字列の組み合わせを認識して別の文字列に置き換える処理を行います。

例えば `{% raw %}【シーン{{.SceneNumber}}】{% endraw %}` と記述したら `{% raw %}{{}}{% endraw %}` で囲まれた部分を認識して（後述の変数を元に） `【シーン1】` などの文字列へと置換します。

.docxをパターンに従って記述することでエピソード内部の各種数字をドキュメントへと反映するのはもちろんのこと、

* シーンやカット繰り返し（ループ処理）
* シーンの始端・終端だけに記述を追加する（条件分岐処理）

など面倒な台本制作の反復作業を自動化できます。

## エピソード変数の確認方法

JSONエクスポートの結果を参照することでどんな変数が利用可能かを確認できます。

## テンプレートの記述方法

### 基本は {{ }} で囲む

### 変数

`{% raw %}{{.SceneNumber}}{% endraw %}`



### 繰り返し

```
{% raw %}{{#Scenes}}
  {{#Scenes.SceneNumber}} or {{.SceneNumber}}
{{/Scenes}}{% endraw %}
```
json上で`[]`でリスト表現されている変数は `{% raw %}{{#   }}{% endraw %}` と記述することで繰り返し処理開始できます。

`{% raw %}{{/   }}{% endraw %}` を記述すると繰り返しを終了します。

繰り返し処理内では、例えばScenesの繰り返し中であれば `{% raw %}{{.SceneNumber}}{% endraw %}` と記述することでシーン番号にアクセスできます。繰り返しの外側では Scenes のどの要素か特定できないため処理できません。

### 条件分岐

`{% raw %}{?{someVar > 5}}...{{:}}...{{/}}{% endraw %}`

json の値で true / false と記述されている変数、あるいは条件式を記述するすることで条件分岐を表現できます。条件がtrue（真）を示す場合は`{% raw %}{{:}}{% endraw %}`の手前が処理され、false（偽）の場合は`{% raw %}{{:}}{% endraw %}`の後ろが処理されます。
